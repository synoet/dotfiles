# synoet's dotfiles
My dotfiles for arch and macos

### dependencies
- [GNU Stow (dotfile manager)](https://www.gnu.org/software/stow/)
- [oh-my-zsh (zsh) ](https://github.com/robbyrussell/oh-my-zsh)
- [picom (compositor) ](https://github.com/yshui/picom)
- [i3-gaps (window manager arch) ](https://github.com/Airblader/i3) 
- [yabai (window manager macos) ](https://github.com/Airblader/i3)
- [polybar (status bar) ](https://github.com/jaagr/polybar) 
- [rofi (launcher) ](https://github.com/davatorium/rofi)
- [neovim (editor) ](https://github.com/neovim)
- [kitty (terminal) ](https://sw.kovidgoyal.net/kitty/)
- [dunst (notifications) ](https://sw.kovidgoyal.net/kitty/)
